## 🚀 Démarrage Rapide

### Pour Utiliser le RAZ
1. Ouvrir l'onglet "RAZ" dans l'application
2. Cliquer sur "Sauvegarder les données" (recommandé)
3. Cliquer sur "Démarrer la RAZ"
4. Sélectionner les options désirées
5. Confirmer et exécuter

### Pour Développer
1. Lire [INSTALLATION-RAZ.md](./INSTALLATION-RAZ.md)
2. Intégrer les composants requis
3. Tester avec [TESTS-VALIDATION-RAZ.md](./TESTS-VALIDATION-RAZ.md)
4. Déployer selon la checklist

### En Cas de Problème
1. Consulter [DEPANNAGE-RAZ.md](./DEPANNAGE-RAZ.md)
2. Utiliser les outils de debug fournis
3. Suivre les procédures de récupération
4. Contacter le support si nécessaire
